# Usage
<pre>
screen -S leakManager
hug -f index.py -p 1337
OR
gunicorn index:__hug_wsgi__ -b 0.0.0.0:1337
OR
uwsgi --http 0.0.0.0:1337 --wsgi-file index.py --callable __hug_wsgi__
ctrl + a + d
</pre>

# Change user/passwd
<pre>
edit etc/LeakManager.conf file
</pre>
